# deepin-cursor-theme

## 安装

### 用源码构建

1. 确保你已经安装了所有需要的编译依赖, 然后执行:
````
sudo apt build-dep .
````
2. 构建:

````
$ mkdir build
$ cd build
$ cmake ..
$ make
````

3. 安装:
````
$ sudo make install
````
